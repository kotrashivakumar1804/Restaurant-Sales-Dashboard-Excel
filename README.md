# Restaurant Sales Dashboard (Microsoft Excel)

## Project Overview
Developed an interactive Excel dashboard to analyze restaurant sales performance using Pivot Tables, Pivot Charts, KPIs, slicers, and advanced Excel functions.

## Features
- Data Cleaning & Validation
- Interactive Dashboard
- KPI Cards
- Pivot Tables
- Pivot Charts
- Slicers
- XLOOKUP
- SUMIFS
- COUNTIF
- IF Functions

## Key Insights
- Total Sales
- Total Orders
- Quantity Sold
- Branch-wise Sales
- Category-wise Sales
- Payment Method Analysis
- Monthly Sales Trends

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- XLOOKUP
- SUMIFS
- COUNTIF
- Data Validation
